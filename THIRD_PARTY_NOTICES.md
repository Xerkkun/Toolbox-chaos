# Third-party notices

This project includes an adaptation of the bifurcation-diagram idea from the user-provided MATLAB code attributed to Lazaros Moysis for the Lorenz system.

Reference used for the adaptation:
- Lazaros Moysis, *Bifurcation diagram for the Lorenz Chaotic system*, MATLAB Central File Exchange.
- The original idea in the supplied code computes a bifurcation / continuation diagram by simulating the Lorenz system, discarding a transient, and recording intersections with a chosen plane.
